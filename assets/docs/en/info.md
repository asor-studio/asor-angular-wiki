## <span id="info-privacy">Privacy Policy</span>

This privacy policy describes how **ASOR Studio (Application Stability & Organizational Research)** handles information related to users who access this documentation.

### Data Collected

This documentation wiki is a client-side static application. **No personal data is collected**, no tracking cookies are used, and no requests are made to external servers for data collection.

### Local Data

The application exclusively uses the browser's **Session Storage** to store the user's language preference (Italian/English). This data:

- Remains exclusively in the user's browser
- Is not transmitted to any server
- Is automatically deleted when the browser session is closed

### Third-Party Libraries

The documentation may load resources from external CDNs (e.g., fonts, icons). These resources are subject to the privacy policies of their respective providers.

### Contact

For privacy-related questions, please contact us through the [Contact](#info-contact) section of this page.
